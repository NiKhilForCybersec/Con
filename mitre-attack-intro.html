MITRE ATT&CK Introduction

Understanding the MITRE ATT&CK framework for detection

What is MITRE ATT&CK?

MITRE ATT&CK (Adversarial Tactics, Techniques, and Common Knowledge) is a knowledge base of adversary behavior based on real-world observations. It's become the standard framework for discussing and mapping threats.

üí° ATT&CK in Simple Terms

ATT&CK answers: "What do attackers actually do after they get in?" It categorizes attack behaviors into:

  • Tactics: The "why" - the attacker's goal (14 tactics)
  • Techniques: The "how" - specific methods to achieve the goal
  • Sub-techniques: More specific variations
  • Procedures: Real-world examples from threat groups

The 14 Tactics

Tactic Goal Example Technique
Reconnaissance Gather information Active Scanning
Resource Development Prepare infrastructure Acquire Infrastructure
Initial Access Get into the network Phishing
Execution Run malicious code PowerShell
Persistence Maintain access Scheduled Task
Privilege Escalation Get higher permissions Valid Accounts
Defense Evasion Avoid detection Obfuscated Files
Credential Access Steal credentials OS Credential Dumping
Discovery Learn the environment System Network Config
Lateral Movement Move through network Remote Services
Collection Gather data to steal Data from Local System
Command & Control Communicate with implants Web Protocols
Exfiltration Steal data Exfil Over C2 Channel
Impact Damage or disrupt Data Encrypted for Impact

Using ATT&CK for SIEM

ATT&CK helps you:

  • Map detections: Tag rules with techniques they detect
  • Identify gaps: What techniques have no coverage?
  • Prioritize: Focus on techniques used by relevant threat groups
  • Communicate: Common language with red team and leadership
üíº Interview Tip

Question: "How do you use MITRE ATT&CK in your detection strategy?"

Answer: "I use ATT&CK for coverage mapping - tagging each detection rule with the techniques it catches. This lets me visualize gaps on the ATT&CK Navigator. I prioritize based on threat intelligence - which techniques are used by groups targeting our industry? I also use ATT&CK to structure purple team exercises - we systematically test detection across the framework. Finally, it's great for communication - I can show leadership our coverage against specific threat groups using ATT&CK heatmaps."
