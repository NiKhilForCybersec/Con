KPI Calculations

KQL: MTTD and MTTR
SecurityIncident
| where TimeGenerated > ago(30d)
| where Status == "Closed"
| extend EventTime = todatetime(FirstActivityTime)
| extend DetectionTime = CreatedTime
| extend ResponseTime = todatetime(FirstModifiedTime)
| extend ClosedTime = ClosedTime
| extend MTTD_hours = datetime_diff('hour', DetectionTime, EventTime)
| extend MTTR_hours = datetime_diff('hour', ClosedTime, DetectionTime)
| summarize 
    AvgMTTD = avg(MTTD_hours),
    AvgMTTR = avg(MTTR_hours),
    P90_MTTD = percentile(MTTD_hours, 90),
    P90_MTTR = percentile(MTTR_hours, 90),
    IncidentCount = count()
  by Severity
| order by Severity
SPL: Incident Volume Trends
index=incidents status=closed
| timechart span=1w count by severity
| addtotals
SPL: SLA Compliance
index=incidents status=closed
| eval sla_target = case(
    severity="Critical", 4,    // 4 hours
    severity="High", 24,       // 24 hours
    severity="Medium", 72,     // 3 days
    severity="Low", 168        // 7 days
)
| eval response_hours = (response_time - created_time) / 3600
| eval sla_met = if(response_hours <= sla_target, 1, 0)
| stats 
    count as total_incidents,
    sum(sla_met) as within_sla,
    avg(response_hours) as avg_response_hours
  by severity
| eval sla_compliance = round(within_sla / total_incidents * 100, 1) . "%"

Detection Coverage

KQL: MITRE Coverage Analysis
SecurityAlert
| where TimeGenerated > ago(90d)
| mv-expand Tactics
| mv-expand Techniques
| summarize AlertCount = count() by 
    Tactic = tostring(Tactics),
    Technique = tostring(Techniques)
| join kind=rightouter (
    externaldata(Tactic:string, Technique:string, TechniqueName:string)
    [@"https://attack.mitre.org/techniques/enterprise"]
) on Tactic, Technique
| extend Coverage = iff(isnotnull(AlertCount), "Covered", "Gap")
| summarize 
    CoveredTechniques = countif(Coverage == "Covered"),
    TotalTechniques = count()
  by Tactic
| extend CoveragePercent = round(CoveredTechniques * 100.0 / TotalTechniques, 1)

Executive Reporting

KQL: Monthly Executive Summary
SecurityIncident
| where TimeGenerated > ago(30d)
| summarize 
    TotalIncidents = count(),
    Critical = countif(Severity == "High"),
    TruePositives = countif(Classification == "TruePositive"),
    AvgResponseHours = avg(datetime_diff('hour', ClosedTime, CreatedTime))
| extend TruePositiveRate = round(TruePositives * 100.0 / TotalIncidents, 1)

ğŸ’¼ Interview Tip

Q: "What metrics do you use to measure SOC effectiveness?"

A: "I focus on metrics that drive improvement:

  1. MTTD (Mean Time to Detect): How quickly do we spot threats? This measures detection capability. Industry average is still 200+ days; goal is hours.
  2. MTTR (Mean Time to Respond): Once detected, how fast do we contain? Critical incidents should be hours, not days.
  3. True Positive Rate: What percentage of alerts are real? Below 60% indicates tuning problems and analyst fatigue.
  4. SLA Compliance: Are we meeting response commitments? Track by severity to ensure critical issues get priority.
  5. Coverage: What can we detect? I map detections to MITRE ATT&CK to identify gaps.
  6. Trending: Are we improving? Month-over-month trends matter more than absolute numbers.

I avoid vanity metrics like 'events processed' that don't indicate security outcomes."
