2. Intelligence-Driven Hunting

Search for specific IOCs or TTPs from threat intelligence:

  • New threat actor TTPs from threat reports
  • IOCs from industry sharing (ISACs)
  • Techniques from recent incidents at peer organizations

3. Baseline-Driven Hunting (Anomaly)

Find deviations from normal behavior:

  • First-time execution of tools
  • Unusual network connections
  • Abnormal user behavior patterns
  • Statistical outliers in any dimension

Hunt Examples

Hunt: Encoded PowerShell Commands

Hypothesis

Attackers use encoded PowerShell to evade basic string-matching detections.

KQL: Hunt Query
SecurityEvent
| where TimeGenerated > ago(7d)
| where EventID == 4688
| where NewProcessName endswith "\\powershell.exe"
| where CommandLine has "-enc" or CommandLine has "-encoded"
| extend DecodedLength = strlen(extract(@"-[eE]nc[^\s]* ([A-Za-z0-9+/=]+)", 1, CommandLine))
| where DecodedLength > 100  // Significant encoded content
| summarize 
    ExecutionCount = count(),
    Hosts = make_set(Computer, 10),
    Users = make_set(SubjectUserName, 10)
  by CommandLine
| order by ExecutionCount desc

Hunt: Rare Process Execution

Hypothesis

Attacker tools are rarely seen across the environment. Finding rare processes may surface threats.

KQL: Hunt Query
SecurityEvent
| where TimeGenerated > ago(7d)
| where EventID == 4688
| extend ProcessName = tostring(split(NewProcessName, "\\")[-1])
| summarize 
    ExecutionCount = count(),
    UniqueHosts = dcount(Computer),
    Hosts = make_set(Computer, 5),
    Users = make_set(SubjectUserName, 5)
  by ProcessName
| where ExecutionCount < 5 and UniqueHosts == 1  // Very rare
| where ProcessName !endswith ".tmp"
| order by ExecutionCount asc

Hunt: Unusual Outbound Connections

Hypothesis

C2 communications may use unusual destination ports or rarely-contacted external IPs.

KQL: Hunt Query
// Find outbound connections to rarely-seen destinations
CommonSecurityLog
| where TimeGenerated > ago(7d)
| where DeviceAction == "allow"
| where DestinationPort !in (80, 443, 53)  // Non-standard ports
| where ipv4_is_private(DestinationIP) == false  // External
| summarize 
    ConnectionCount = count(),
    SourceHosts = dcount(SourceIP),
    Ports = make_set(DestinationPort)
  by DestinationIP
| where SourceHosts == 1 and ConnectionCount < 10  // Rare
| order by ConnectionCount asc

Hunt: Service Account Anomalies

Hypothesis

Compromised service accounts may show interactive logons or access to unusual systems.

KQL: Hunt Query
// Service accounts with interactive logons
SecurityEvent
| where TimeGenerated > ago(7d)
| where EventID == 4624
| where LogonType in (2, 10)  // Interactive, RemoteInteractive
| where Account has "svc_" or Account has "service"
| summarize 
    LogonCount = count(),
    Hosts = make_set(Computer, 10),
    SourceIPs = make_set(IpAddress, 10)
  by Account
| order by LogonCount desc

Hunt: Scheduled Task Persistence

Hypothesis

Attackers create scheduled tasks for persistence. Tasks with suspicious commands may indicate compromise.

KQL: Hunt Query
SecurityEvent
| where TimeGenerated > ago(30d)
| where EventID == 4698  // Scheduled task created
| extend TaskContent = tostring(parse_xml(EventData).DataItem.TaskContent)
| where TaskContent has_any (
    "powershell", "cmd.exe", "wscript", "cscript",
    "mshta", "rundll32", "regsvr32",
    "http://", "https://",
    "-enc", "-encoded"
)
| project TimeGenerated, Computer, SubjectUserName, TaskName, TaskContent
| order by TimeGenerated desc
