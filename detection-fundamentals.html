Detection Fundamentals

Core concepts of security detection engineering

What is Detection Engineering?

Detection engineering is the practice of designing, building, testing, and maintaining detection rules that identify security threats in your environment.

ğŸ’¡ The Detection Pyramid

Effective detection requires:

  • Visibility: The right logs collected
  • Context: Understanding what's normal
  • Logic: Rules that identify threats
  • Tuning: Reducing false positives
  • Response: Actionable alerts

Detection Types

Type How It Works Example
Signature-based Match known-bad patterns Specific malware hash, known C2 IP
Behavioral Detect actions regardless of tool Process creating scheduled task + network connection
Threshold Count exceeds limit 5+ failed logins in 5 minutes
Anomaly/ML Deviation from baseline User accessing unusual systems
Correlation Multiple events together Failed login â†’ success â†’ privilege escalation

Detection Quality Metrics

  • True Positive Rate: How often it catches real threats
  • False Positive Rate: How often it fires on benign activity
  • Coverage: What percentage of attack techniques it detects
  • Timeliness: How quickly it alerts after the event
